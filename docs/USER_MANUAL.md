## 产品介绍

游戏加速器是一款专业的网络加速工具，专为游戏玩家设计。通过优化的网络传输路径和智能节点选择，有效降低游戏延迟，提升游戏体验。

### 主要特性

- ✅ **智能加速**: 自动识别游戏进程，智能选择最优加速节点
- ✅ **多平台支持**: 支持Steam、Epic、Origin、Battle.net等主流游戏平台
- ✅ **低延迟**: 数据转发延迟控制在50ms以内
- ✅ **安全加密**: 采用AES-256加密算法保护数据传输
- ✅ **自动重连**: 断线自动重连，确保连接稳定
- ✅ **实时监控**: 实时显示连接状态、流量统计等信息

## 快速开始

### 第一步：注册账户

1. 启动游戏加速器客户端
2. 点击"注册账户"按钮
3. 填写用户名、邮箱和密码
4. 点击"注册"完成账户创建

### 第二步：登录

1. 在登录界面输入用户名和密码
2. 点击"登录"按钮
3. 登录成功后进入主界面

### 第三步：选择节点

1. 在主界面选择"地区"
2. 从节点列表中选择延迟最低的节点
3. 或选择"自动选择"让系统智能分配

### 第四步：开始加速

1. 点击"连接"按钮
2. 等待连接建立（状态显示"已连接"）
3. 启动游戏，享受加速服务

## 功能详解

### 1. 智能节点选择

系统提供多种节点选择策略：

- **自动选择**: 系统自动选择延迟最低的节点
- **地区选择**: 按地区筛选节点
- **手动选择**: 从节点列表中手动选择

### 2. 游戏进程识别

支持自动识别以下游戏平台：

| 平台 | 支持游戏示例 |
|------|--------------|
| Steam | CS2, Dota2, PUBG等 |
| Epic Games | Fortnite, GTA5等 |
| Battle.net | 魔兽世界, 守望先锋等 |
| Origin | Apex Legends, FIFA等 |
| Riot Games | 英雄联盟, Valorant等 |
| Ubisoft | 彩虹六号, 刺客信条等 |

### 3. 代理模式

程序支持两种代理模式：

- **SOCKS5代理**: 默认端口1080
- **UDP代理**: 默认端口1081

游戏客户端需要配置代理指向本地代理端口。

### 4. 系统托盘

程序最小化后会驻留在系统托盘：

- 双击托盘图标显示主窗口
- 右键托盘图标显示快捷菜单
- 支持后台运行

## 常见问题

### Q1: 连接失败怎么办？

**可能原因及解决方案：**

1. **网络问题**
   - 检查网络连接是否正常
   - 尝试切换网络环境

2. **防火墙拦截**
   - 将程序添加到防火墙白名单
   - 允许程序通过防火墙

3. **服务器维护**
   - 稍后重试
   - 联系客服确认服务器状态

### Q2: 加速效果不明显？

**优化建议：**

1. 选择延迟更低的节点
2. 关闭其他占用带宽的程序
3. 检查本地网络质量
4. 尝试重启加速器

### Q3: 游戏无法连接？

**排查步骤：**

1. 确认加速器已连接
2. 检查游戏代理设置
3. 确认游戏端口未被阻止
4. 尝试切换加速节点

### Q4: 程序崩溃怎么办？

**解决方案：**

1. 重启程序
2. 清除配置文件（~/.game_accelerator/）
3. 重新安装程序
4. 联系技术支持

### Q5: 如何查看日志？

日志文件位置：
- Windows: `C:\Users\<用户名>\.game_accelerator\logs\`
- macOS: `~/.game_accelerator/logs/`
- Linux: `~/.game_accelerator/logs/`
